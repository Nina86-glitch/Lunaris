# Lunaris — Лунный путеводитель 🌙

## Как установить на iPhone (бесплатно)

### Шаг 1: Создайте аккаунт на GitHub
Зайдите на [github.com](https://github.com) и зарегистрируйтесь (бесплатно).

### Шаг 2: Создайте репозиторий
1. Нажмите **"+"** → **"New repository"**
2. Назовите его `lunaris`
3. Поставьте галочку **"Public"**
4. Нажмите **"Create repository"**

### Шаг 3: Загрузите файлы
1. Нажмите **"uploading an existing file"**
2. Перетащите все 5 файлов из этой папки:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Нажмите **"Commit changes"**

### Шаг 4: Включите GitHub Pages
1. Перейдите в **Settings** → **Pages**
2. В разделе **Source** выберите **"main"** branch
3. Нажмите **Save**
4. Через 1-2 минуты ваш сайт будет доступен по адресу:
   `https://ваш-логин.github.io/lunaris/`

### Шаг 5: Установите на iPhone
1. Откройте ссылку в **Safari** на iPhone
2. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой внизу)
3. Выберите **«На экран Домой»**
4. Нажмите **«Добавить»**

Готово! Приложение будет на домашнем экране как обычное приложение.
